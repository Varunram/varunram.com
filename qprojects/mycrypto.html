MyCrypto

MyCrypto is an open source web3.0 blockchain interface similar in function to MyEtherWallet and is founded by the ex-lead developer of MyEtherWallet. It has the same goals as MyEtherWallet in wanting to provide an easy interface to interact with Ethereum style blockchains, while providing and intuitive and easy to use UI.

MyCrypto's frontend UI uses React.js and its desktop application uses the electron.js framework.

My contributions to MyCrypto are mostly related to fixing backend bugs, small frontend UI bugs, and issue reporting. I also integrated Musicoin support to MyCrypto, when MyCrypto launched in late June. I also love interacting with the cheerful team at MyCrypto who are never uneager to help and still continue to be a contributor to MyCrypto, reporting bugs when I encounter them and fixing them if possible.
