MyCrypto

MyCrypto is a blockchain interface similar to MyEtherWallet and is headed by ex-lead developers of MyEtherWallet. It has the same goal as MyEtherWallet in wanting to provide an easy interface to interact with Ethereum style blockchains.

MyCrypto's website is written in javascript and its desktop application written using the electron.js framework.

My contributions to MyCrypto are mostly related to bug fixing and issue reporting besides adding Musicoin support to MyCrypto. I love interacting with the cheerful team at MyCrypto who are never uneager to help and still continue to be a contributor.
