MyCrypto

MyCrypto is a blockchain interface similar to that of MyEtherWallet and is headed by the once lead developers of MyEtherWallet. It seeks to achieve something similar to MyEtherWallet by wanting ot provide an easy interface to interact with Ethereum style blockchains using the web3.js framework

MyCrypto has a website written using javascript and a desktop application written using electron.js.

My contributions to MyCrypto were mostly related to bug fixing and reporting besides adding Musicoin support to MyCrypto. I loved interacting with the cheerful team at MyCrypto who are never unavailable for help. I still continue to regularly use and report bug fixes in MyCrypto and report bugs in their implementation.
