Metabase

Metabase is a database visualization framework that can be used with a variety of databases. There are tons of database frameworks and it is difficult to have UIs designed specifically for all of them. This is where metabase comes in, providing easy access queries with UI buttons. Metabase also provides data analysis tools that can be used to analysze the database.
